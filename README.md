## 🔭 Active projects

### AutomaticFlow `Python`
AI workspace manager for organizations and freelancers. Handles agent task routing, context management, and multi-user session isolation.

### AI Content Studio `JavaScript`
End-to-end pipeline for AI image generation and editing, with direct publish integrations to social networks. Built for non-technical content teams.

### AI Orchestrator Desktop `JavaScript`
Desktop shell for coordinating multiple AI agents and workflows locally. Targets power users who need persistent, offline-capable AI sessions.

### WhisperDesk `Python`
Local audio transcription app with automatic speaker diarization. Fully offline — no data leaves the device. Built on Whisper + pyannote.audio.

### PageWhisper `TypeScript · Chrome MV3`
Browser extension that extracts web components and converts them to clean, production-ready code using AI. Manifest V3 compliant.

### saas-multi-tenant `TypeScript`
Multi-tenant SaaS platform with pluggable vertical modules: eCommerce, Events, Real Estate, Restaurants. Shared infrastructure, isolated tenant data.

### wp-custom-api `PHP · WordPress`
WordPress plugin exposing a fully configurable REST API surface with admin dashboard. No hardcoded endpoints — schema-driven configuration.

### DevDock `Tauri v2 · React`
Developer project launcher built on Tauri v2. Fast, native-feeling dev environment switcher. (Fork of yosnap/devdock.)
